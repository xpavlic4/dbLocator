Rest API Resources

JDBCs

Oracle JDBC connection strings.

JDBC

Resource

GET jdbcs/:country

Description

Show JDBC connection string for given country.

Example: http://localhost:8080/DbLocator/rest/databases/jdbcs/at

<jdbc>
	<connectionString>jdbc:oracle:thin:@//at1.focusmr.co.at:1521/aemAT1.FOCUSMR</connectionString>
	<country>at</country>
</jdbc>
